# 河南洛河畔律师事务所官方网站

Henan Luohepan Law Firm Official Website

## 🌟 项目特点

- **高端设计风格**: 采用深色主题配合金色点缀，体现专业权威形象
- **响应式设计**: 完美适配桌面端、平板和手机设备
- **双语支持**: 完整的中英文切换功能，服务国际客户
- **流畅动画**: 使用 Framer Motion 实现优雅的页面动效
- **现代化技术栈**: React 18 + TypeScript + Vite + Tailwind CSS

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

## 📁 项目结构

```
src/
├── components/          # React 组件
│   ├── Navbar.tsx      # 导航栏
│   ├── Hero.tsx        # 首屏英雄区
│   ├── Services.tsx    # 执业领域
│   ├── Team.tsx        # 律师团队
│   ├── Cases.tsx       # 成功案例
│   ├── Testimonials.tsx # 客户评价
│   ├── About.tsx       # 关于我们
│   ├── Contact.tsx     # 联系我们
│   └── Footer.tsx      # 页脚
├── i18n/               # 国际化配置
│   ├── config.ts       # i18n 配置
│   └── locales/        # 语言文件
│       ├── zh-CN.json  # 中文
│       └── en-US.json  # 英文
├── styles/             # 样式文件
│   └── index.css       # 主样式
├── App.tsx             # 主应用组件
└── main.tsx            # 入口文件
```

## 🎨 设计特色

### 配色方案
- **主色**: 深海军蓝 (#0a1628) - 体现专业与信任
- **强调色**: 金色 (#c9a962) - 彰显高端品质
- **辅助色**: 中性灰系列 - 确保可读性

### 字体
- **标题**: Noto Serif SC / Playfair Display - 优雅专业
- **正文**: Inter - 现代简洁

### 核心页面

1. **Hero 区域**: 震撼的首屏展示，包含关键统计数据
2. **执业领域**: 6大核心法律服务领域展示
3. **律师团队**: 资深律师团队介绍
4. **成功案例**: 代表性案例展示
5. **客户评价**: 客户真实反馈
6. **关于我们**: 律所价值观与国际合作网络
7. **联系我们**: 完整的联系表单和地图
